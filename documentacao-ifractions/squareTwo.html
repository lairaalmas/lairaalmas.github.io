GAME LEVELS - SQUARE III: fraction comparisson level Name of game state : 'squareTwo' Shape : square Character : kid Theme : (not themed) Concept : player select equivalent dividends for fractions with different divisors Represent fractions as : subdivided blocks # of different difficulties for each level : 5 Level : 'C' (in variable 'levelType') C : Player selects equivalent fractions of both blocks Sublevels can be : 'B' or 'C' (in variable 'sublevelType') B : equivalence of fractions top has more subdivisions C : equivalence of fractions bottom has more subdivisions
