Personalização

O iFractions dispõe de uma coletânea de jogos que abordam diferentes assuntos relacionados a frações. O usuário pode personalizar o jogo como desejado nos menus do iFractions. Uma vez configurado, o ele gera um mapa de fases para o jogo atual, com nível crescente de dificuldade. O objetivo do usuário é chegar ao final do mapa, finalizando todos os jogos.

Atualmente, os jogos são gerados a partir de uma combinação de: jogo, modo de jogo, operação matemática, nível de dificuldade e elementos visuais auxiliares. A figura \ref{fig:exjogo}, apresenta um exemplo da primeira fase do mapa gerado utilizando a configuração do jogo presente na tabela \ref{tab:confjogo}.

  • Jogo (gameType)

    É o conceito principal. Temos disponíveis atualmente 3 jogos: quadriláteros I e quadriláteros II, que lidam com frações aplicadas em figuras retangulares, e círculos I, que apresenta fração como um arco.

  • Modo de jogo (gameMode)

    São as diferentes variações para o jogo selecionado.

  • Operações matemáticas (gameOperation)

    Representam diferentes conteúdos de frações (soma, subtração, igualdade etc) que podem ser abordados pelo mesmo jogo.

  • Nível de dificuldade (gameDifficulty)

    Representado como um número inteiro tal que quando maior o valor, maior a dificuldade, informa a dificuldade geral do jogo, sendo parâmetro para criação dos 4 fases do jogo selecionado.

  • Elementos visuais auxiliares (fractionLabels )

    Dizem respeito ao controle da adição ou remoção de elementos visuais que podem facilitar o jogo, como mostrar ou não as frações na tela.
